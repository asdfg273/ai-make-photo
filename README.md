AI Make Photo (本地极简版 AI 绘画)

这是我自己做的一款本地运行的 AI 绘画工具。相比于庞大复杂的 WebUI，这个版本更轻量、不卡电脑、没有花里胡哨的配置，直接双击运行就能画图！支持全中文输入，还内置了超好用的修图功能。

⚙️ 怎么跑起来？(安装步骤)
把这个项目下载到你的电脑上。
建议用 Python 3.10 左右的版本。在项目文件夹里打开终端（黑框框），输入下面这行命令安装依赖：
bash
复制代码
收起
pip install -r requirements.txt
复制
(注意：如果画图太慢，说明你的 PyTorch 没装 GPU 版，去 PyTorch 官网找对应的 CUDA 安装命令重新装一下 torch 即可)
运行程序：
bash
复制代码
收起
python main.py
复制
📂 必须要建的文件夹！(非常重要)

为了防止模型冲突报错，一定要在项目文件夹里建好这几个目录，并把下载好的模型（.safetensors 文件）按规定放进去：

📁 models/ —— 放基础大模型（比如 ChilloutMix、SDXL_Base 等）。
📁 loras/
📁 sd1.5/ —— 放 SD 1.5 专用的 LoRA（动漫/写实风通常是这个）。
📁 sdxl/ —— 放 SDXL 专用的巨大 LoRA。
📁 photo/ —— 画好的图会自动保存在这里。

(可以在模型旁边建一个同名的 .txt 文件写上提示词或者心得，软件里会自动读取显示哦！)

🕹️ 怎么玩？(功能说明)
1. 基础画图 (傻瓜式操作)
左上角选好你想用的 主模型。
输入提示词：支持全中文！（比如打一句：一个帅气的赛博朋克男生，在下雨的街头，高质量），软件会在后台自动帮你翻译成英文发给 AI。
点底部的 [开始生成] 即可。
2. 智能 LoRA 换画风
当你选了主模型后，下面的 LoRA 列表会自动过滤！选了 XL 的大模型，下面就只显示 XL 的 LoRA，绝对不会选错报错。
选好 LoRA 后，旁边的滑块可以调强度（一般拉到 0.6 - 0.8 效果最好）。
3. 涂鸦改图 (重点推荐)

点左边的 [📂 打开历史图库并编辑]，选中你之前画过的图。
弹出修图面板后，用 [画笔] 把你想改的地方（比如衣服、背景）涂抹掉。
点 [✅ 保存并传回主界面]。
这时候原图和遮罩已经自动过去啦，你改一下提示词（比如改成 穿红色外套），重绘幅度拉到 0.6 左右，点生成，直接无缝换装！
修图面板里还能直接给图片加字，字体能随便拖动和缩放！
4. 自动修脸 (ADetailer)

画全身照的时候，人脸很容易变成“邪神”。
只要勾选主界面上的 [启用 ADetailer 脸部修复]，出图后软件会在后台自动把脸部抠出来重新精修，完全不用你管。

5. 自动画连环画 (分镜功能)

在提示词里用中括号把词括起来，比如输入 [苹果, 香蕉, 橘子] 放在桌子上。
软件会自动识别，帮你连续画 3 张图，最后还会自动拼成一张漫画九宫格大图！
